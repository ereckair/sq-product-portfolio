The technical edge

Three layers, routed intelligently

Rather than sending every message to a large model, the agent routes by complexity. That's where the speed and the ~80% cost saving come from.

LAYER 1

Rule hard-routing

High-frequency commands answer in milliseconds without touching an LLM.

/userlist · generate PPT · view images
LAYER 2

Model routing

Simple intents go to a small, cheap model; only complex reasoning reaches the main model.

simple → mini (70%) · complex → main (30%)
LAYER 3

LLM agent + tools

Full intent recognition and tool-calling across nine intent categories for everything else.

tool calling · 9 intent categories
